### git config
Global setting
*git config --global --list*

Local setting
*git config --list*

### Initialize 
*git init*

### Add
*git add <filename>*
*git add .*  All files

### Commit
*git commit -m "<COMMIT_MSG>"*

### Remote Origin Add
*git remote add origin <URL>*

### Origin Push
*git push -u origin master*
